9、Mac自动补全功能

（1）进入终端输入：nano .inputrc，然后回车

（2）复制以下内容到文件里

```git
set completion-ignore-case on
set show-all-if-ambiguous on
TAB: menu-complete
```

（3）按下control+o，然后再回车，重启终端即可
